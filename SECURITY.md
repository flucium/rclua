# Security Policy
We will make a reasonable effort to provide a fix within 90 days whenever possible. However, the time required may vary depending on the nature and complexity of the issue.

## Supported Versions
Only the latest version is supported.

## Reporting a Vulnerability
If you discover a vulnerability, please report it privately. Do not disclose it publicly.

Please : [Security advisory](https://github.com/flucium/rclua/security/advisories/new)
